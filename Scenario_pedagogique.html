Scénario pédagogique

Fondamental : Conducteur pédagogique

Le conducteur pédagogique permet de formaliser un déroulé de formation en définissant :

  • Les objectifs ;

  • Les compétences visées ;

  • Les modalités de réalisation d'un parcours d'apprentissage.

Qu'il s'agisse d'un dispositif :

  • En présentiel ;

  • À distance ;

  • Ou hybride[1].

Le conducteur décrit également :

  • Les activités à réaliser ;

  • Les consignes d'exécution pour l'apprenant et pour le formateur ;

  • Les supports pédagogiques associés.

Les scénarios réalisés peuvent être publiés :

  • En 2 versions : l'une pour le Formateur et l'autre pour l'Apprenant ;

  • En 3 formats : Web, PDF, ODT (pour relecture).

Complément : Pourquoi

Le conducteur pédagogique vise en premier lieu à formaliser un scénario d'apprentissage :

  • pour structurer ses propres cours ;

  • pour le partager avec d'autres formateurs ou tuteurs ;

  • pour le diffuser aux apprenants.

Il permet également de recenser les supports pédagogiques nécessaires à ce scénario.

Méthode : Comment

Le guide d'écriture[2] du Conducteur Pédagogique vous permet de construire votre scénario pédagogique en définissant pour chaque activité :

  • Les prérequis et les objectifs ;

  • Les compétences visées ;

  • Les modalités de réalisation  :

    • Durée pour l'apprenant

    • Mode présentiel ou à distance, synchrone ou asynchrone

    • Travail individuel ou collaboratif

  • Le typage de l'activité, selon une liste préétablie basée sur la nomenclature LOM-FR (apprendre, observer, produire, s'évaluer...) ou personnalisée grâce à l'item "entrée d'activité" ;

  • Les consignes, pour l'apprenant et le formateur ;

  • Le matériel pédagogique, pour l'apprenant et le formateur ;

Des ressources peuvent également être intégrées dans chaque activité et sous-activité :

  • Des supports pédagogiques réalisés avec Opale au format Web, Papier ou Diaporama ;

  • Des documents bureautiques.

Le Conducteur pédagogique vous permet de scénariser des modules Opale et/ou des ressources produites avec d'autres chaines éditoriales ou d'autres logiciels.

Exemple :

Le guide pédagogique que vous avez devant vous au format papier est une version pour l'apprenant.

Fondamental : Alignement pédagogique

Les activité d'apprentissage doivent être alignées avec les compétences à développées et les objectifs à atteindre.

Exemple :

Dans le TP « Identification des lipides » l'alignement pédagogique est illustré dans le tableau ci-dessous :

Alignement pédagogique

Objectifs

Connaître comment identifier les lipides

Être capable de confirmer la présence ou l'absence des lipides dans les différents échantillons.

Être capable d'identifier la présence des lipides en tant que composant des cellules végétales.

Prérequis

  1. Connaître les différents types des lipides ;

  2. Être capable d'utiliser les différents types des lipides ;

  1. Connaître comment mesurer le volume ;

  2. Être capable de mesurer un volume ;

  3. Être capable de mesurer et déplacer des solutions liquides.

  1. Connaître l'utilité d'un microscope optique  ;

  2. Être capable de manipuler un microscope optique ;

  3. Être capable d'observer microscopiquement des préparations biologiques.

Vérification des prérequis

Exercice 1 : Types des lipides

Exercice 2 : Manipulation de la pipete et la poire.

Exercice 3 : Manipulation du microscope.

Activités d'apprentissage

Problème 1 : Test du papier

Problème 2 : Coloration Sudant III

Problème 3 : Lipides au niveau cellulaire

Évaluation des apprentissage

Question 1 : Peut-on comparer la quantité des lipides dans un aliment à l'issu du test de la tache translucide ? discutez.

Question 5 : Vous êtes appelés à contrôler la qualité d'un produit alimentaire quel méthode adopter. Comment pouvez-vous dire?

Question 2 : Dans quel contexte pouvez vous utilisez les test de Sudan III et pour quel objectif ?

Question 3 : Lequel des trois échantillons contenait des lipides et en quelles quantités ?

Question 4 : Les lipides étaient ils uniformément répartis dans les cellules ou localisés ?

Notes

  1. Hybride

    Un dispositif de formation hybride intègre des séquences en présentiel et d'autres à distance, avec des activités à réaliser en mode synchrone et/ou asynchrone.

  2. Guide d'écriture

    Le guide d'écriture est un ensemble de formulaires préconçus, chacun correspondant à un item du modèle documentaire.

    Chaque formulaire s'affiche dans la zone d'édition lorsqu'un item est ouvert. Il guide l'auteur dans la rédaction et la structuration du contenu : il est composé de rubriques, blocs de texte, balises d'intention, métadonnées, options,...
